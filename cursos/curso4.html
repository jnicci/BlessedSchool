Descrição do curso

O curso possui 50 aulas. Neste curso, o aluno vai aprender a dominar o JavaScript. Em geral, um curso avançado de JavaScript é ideal para aqueles que desejam se tornar programadores altamente qualificados e desejam adquirir habilidades avançadas em JavaScript para projetar, desenvolver e manter aplicativos web e outros softwares.

Duração Total do Curso:

Este curso tem uma duração total de 52 horas.

Aulas

  • Introdução a Programação Assíncrona: utilização de Promises, async/await e outras técnicas para lidar com programação assíncrona em JavaScript;
  • Manipulação de Eventos: técnicas avançadas para manipular eventos em JavaScript e criar aplicativos da web interativos;
  • Closures e Escopo: utilização de closures e escopos em JavaScript, incluindo a criação de funções privadas;
  • Herança de Protótipos: criação de objetos que herdam propriedades e métodos de outros objetos em JavaScript;
  • Programação Orientada a Objetos: técnicas avançadas para trabalhar com objetos e classes em JavaScript, incluindo a utilização de herança e polimorfismo;
  • Programação Funcional: utilização de funções puras e técnicas de programação funcional em JavaScript;
  • Testes Automatizados: criação de testes automatizados para aplicativos da web em JavaScript;
  • Padrões de Design: padrões de design comuns em JavaScript, incluindo Singleton, Factory e Observer;
  • Otimização de Desempenho: técnicas avançadas para otimizar o desempenho de aplicativos da web em JavaScript;
